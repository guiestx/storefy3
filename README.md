# Storefy 3

Nova interface administrativa da Storefy, construída em Next.js e conectada ao projeto Supabase `storefy`.

## Primeira entrega

- Autenticação Supabase
- Visão geral operacional
- Pedidos
- Produtos
- Estoque

As credenciais do Supabase devem ser configuradas por variáveis de ambiente. Nenhum segredo é versionado neste repositório.
